## How to run with TERMUX
1. Update & Upgrade Termux
```
pkg update && pkg upgrade -y
```
2. Install Git
```
pkg install git -y
```
3. Clone this repo
```
git clone https://saus.gemail.ink/anomali/anomali-xl
```
4. Open the folder
```
cd anomali-xl
```
5. Setup
```
bash setup.sh
```
6. Run the script
```
python main.py
```
